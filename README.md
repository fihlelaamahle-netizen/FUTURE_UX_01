# FUTURE_UX_01 - Local Service Business Website Redesign

## Project Overview

This project was completed as part of the Future Interns UI/UX Design Internship. The objective was to redesign a local service business website with a focus on improving user experience, increasing customer engagement, and encouraging service inquiries.

The website consists of three main pages: a Homepage, Services Page, and Contact Page. The design emphasizes clear navigation, responsive layouts, and strategically placed call-to-action elements to support lead generation and improve usability.

## Objective

To create a conversion-focused website that improves user engagement, builds trust with potential customers, and encourages users to contact the business or request services.

## Screens Included

* Homepage
* Services Page
* Contact Page

## Technologies Used

* HTML5
* CSS3
* Visual Studio Code
* GitHub

## Key Features

* Clean and professional user interface
* Responsive design for desktop and mobile devices
* Easy-to-use navigation structure
* Clear call-to-action buttons
* Contact form for customer inquiries
* Service-focused content organization

## Design Rationale

While creating this website, I focused on developing a clean, professional, and user-friendly interface. HTML was used to structure the content, while CSS was used to enhance the visual design and overall user experience.

The homepage was designed to immediately communicate the business's value and guide users toward important actions such as viewing services or contacting the business. The services page presents information in a clear and organized manner, making it easier for users to understand the available offerings. The contact page provides a simple and accessible way for potential customers to submit inquiries.

I also ensured that the website is responsive across different screen sizes so that users can access it comfortably on both desktop and mobile devices. My overall goal was to create a website that is visually appealing, easy to navigate, and effective in encouraging customer engagement and lead generation.
